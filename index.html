🚀 배포 후 구글/네이버 등록 (5분)
🔵
① GitHub Pages 배포
다운로드한 HTML 파일을 index.html로 이름 바꿔서 GitHub에 업로드
1
github.com → New repository → pr-article-studio
2
Add file → Upload files → index.html 업로드
3
Settings → Pages → Branch: main → Save
4
jhkim-afk.github.io/pr-article-studio 접속 확인
무료완료까지 3분
🔴
② 구글 서치콘솔 등록
배포 완료 후 구글에 URL 등록하면 3~7일 내 검색 노출
1
search.google.com/search-console 접속
2
URL 접두어 → 배포된 URL 입력
3
소유권 확인 (HTML 파일 방식 추천)
4
URL 검사 → 색인 생성 요청 클릭
무료3~7일 내 노출
🟢
③ 네이버 서치어드바이저
네이버 검색 노출을 위해 사이트 등록 필수
1
searchadvisor.naver.com 접속
2
웹마스터 도구 → 사이트 등록
3
HTML 태그 방식으로 소유확인
4
요청 → 웹 페이지 수집 → URL 제출
무료1~2주 내 노출
📣
④ SNS 링크 공유
배포 URL을 SNS에 공유하면 크롤링 속도가 빨라지고 백링크 효과도 있어요
1
링크드인에 기사 URL + 한줄 요약 포스팅
2
페이스북 페이지에 공유
3
카카오톡 채널 / 단톡방 공유
4
트위터/X 공유
크롤링 가속백링크 효과
⏱️ 예상 타임라인: 배포 → 즉시 | 구글 노출 → 3~7일 | 네이버 노출 → 1~2주 | 키워드 상위 노출 → 2~4주
